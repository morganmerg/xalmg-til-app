# Хальмг Тиль — мобильное приложение

Кросс-платформенное (iOS + Android) приложение для изучения калмыцкого языка.
Спутник гранта [[20_Areas/ХальмгТиль/grant_application|«Хальмг Тиль»]] —
ориентир тот же (500 школьников Элисты), но канал поставки другой: мобильное
устройство, оффлайн.

## Где что

- **README.md** — быстрый старт, запуск, ребилд данных
- **docs/superpowers/specs/2026-04-21-xalmg-til-app-design.md** — дизайн-спец
- **scripts/** — pipeline словаря: DSL → JSON → SQLite
- **src/** — приложение

## Источники данных

- Словарь: `xal-rus.dsl.dz` (Lingvo, 22 846 статей, UTF-16 LE)
- Аудио: `xal-rus-audio/` (366 WAV, имена в мёртвой кодировке)
- Пит-дизайн: `scenes.jsx`, `animations.jsx`, `ios-frame.jsx` — уже содержат
  степную палитру, орнамент, билингва-UI и layout главного экрана и урока
