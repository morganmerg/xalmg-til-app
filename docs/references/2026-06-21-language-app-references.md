## Предварительные выводы для Хальмг Тиль

1. Главный экран не должен быть карточной свалкой. Нужен понятный daily route: сегодня выучи 5 слов, повтори 3, открой слово дня.
2. Словарь — сильная сторона текущего MVP. Его надо сделать центральной вкладкой, но визуально легче: быстрый поиск, история, избранное, примеры.
3. Уроки лучше строить не как "тест ради теста", а как короткую сессию: увидеть, услышать, выбрать, произнести, повторить.
4. Национальный слой должен быть системным: палитра флага + лотос + орнаментальная сетка + мотив степи/гера/костюма. Не сувенирная открытка.
5. Для Figma стоит делать не один экран, а минимальный screen set: onboarding, home, dictionary search, entry detail, lesson question, lesson result, profile/progress.
