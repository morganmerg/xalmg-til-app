## Styles

- Style A, clean modern learning: strongest product foundation. Use its spacing, card hierarchy, bottom nav, and restrained palette.
- Style B, cultural editorial: strongest cultural identity, but too ornate in places. Useful for borders, lotus/zala accents, and premium tone.

## Prompt Rules Used

- Generate one screen per image, not a multi-screen sheet.
- Use Kalmykia-inspired blue, yellow, red, black, and gold.
- Use restrained lotus/zala and Kalmyk costume-inspired collar/trim references.
- Include the saiga calf mascot where useful, with juvenile horn buds and a broad saiga nose.
- Treat all generated text as placeholder; final text must be rebuilt manually in Figma.
- Avoid Kazakh loop ornaments, generic yurt/carpet motifs, exact official emblem copies, and direct Duolingo cloning.
